## 四、关键技术依赖关系

```
佣金计算引擎（上篇）
  ↕ 分润链路递归调用
多级结算政策（补充A）— override_hierarchy JSON → commission_split 表

车险政策管理（补充A）
  ↕ 报价前置校验
预核保禁保名单 → Redis缓存 → 报价引擎实时拦截

薪酬管理（补充B）
  ↕ 数据聚合来源
commission_record（上/中篇）+ comm_salary_adjustment（补充B）→ 工资查询聚合

非车系统设置（补充B）
  ↕ 保单导入联动
non_motor_import_template → 非车保单导入（PDF-088）解析引擎
non_motor_field_config → 非车保单录入表单动态渲染（前端）
```
