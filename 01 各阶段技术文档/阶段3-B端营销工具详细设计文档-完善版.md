## 二、营销素材模块详细设计

### 2.1 海报模板管理

#### 2.1.1 功能概述
海报模板管理是营销工具的核心功能,为代理人提供丰富的海报模板资源,支持快速生成个性化营销海报。

#### 2.1.2 核心业务逻辑

**模板分类体系**:
- **节日海报**: 春节、端午、中秋等传统节日主题
- **产品海报**: 车险、寿险、健康险等产品推广
- **活动海报**: 促销活动、新品发布等营销活动
- **通用海报**: 品牌宣传、服务介绍等常规场景

**模板元素设计**:
1. **可编辑文本元素**: 标题、副标题、价格、联系方式等
   - 每个文本元素包含:位置坐标(x,y)、字体大小、颜色、字体类型、最大长度
   - 支持多行文本和自动换行
   - 文本超长时自动缩小字号或截断

2. **可替换图片元素**: 产品图、人物照片等
   - 定义图片区域:位置(x,y)和尺寸(width,height)
   - 支持图片裁剪、缩放适配
   - 保持图片比例或强制拉伸

3. **固定二维码位置**: 个人推广二维码
   - 固定位置和尺寸配置
   - 自动生成包含邀请码的二维码
   - 支持二维码Logo嵌入

**模板使用统计**:
- 每次生成海报,模板使用次数+1
- 按使用频率排序模板列表
- 为运营提供模板效果数据

**海报生成核心业务流程**:

1. **参数验证阶段**:
   - 校验模板ID是否存在且状态为启用
   - 校验用户上传的自定义文本是否超长
   - 校验用户上传的自定义图片是否符合尺寸要求

2. **二维码生成阶段**:
   - 生成用户唯一邀请码(如果未生成)
   - 构建邀请链接: `https://h5.domain.com/invite?code={inviteCode}`
   - 使用ZXing库生成二维码图片
   - 二维码中心可选嵌入企业Logo

3. **图层合成阶段**:
   - 加载模板背景图片作为底图
   - 按照元素配置依次绘制:
     * 自定义图片元素(裁剪、缩放后绘制)
     * 文本元素(计算文本宽度、自动换行、居中对齐等)
     * 二维码元素(绘制到指定位置)
   - 可选添加水印防盗用

4. **文件保存阶段**:
   - 将合成的图片保存到临时目录
   - 上传到OSS对象存储(阿里云/七牛云)
   - 获取图片访问URL
   - 删除临时文件

5. **记录保存阶段**:
   - 保存用户生成海报记录到`marketing_user_poster`表
   - 更新模板使用次数
   - 返回海报图片URL给前端

**关键技术点**:
- 使用Java AWT或ImageMagick进行图片合成
- 中文字体需要加载TTF字体文件
- 图片合成建议异步处理,避免接口超时
- 生成的海报添加唯一标识水印防盗用

### 2.2 文案库管理

#### 2.2.1 功能概述
为代理人提供精选营销文案,支持一键复制使用,提升营销效率。

#### 2.2.2 核心业务逻辑

**文案分类体系**:
- **产品文案**: 按险种分类(车险、寿险、健康险、意外险等)
- **场景文案**: 按营销场景分类(开门红、促销活动、节日营销等)
- **朋友圈文案**: 适合朋友圈发布的短文案
- **短信文案**: 适合短信群发的模板文案

**文案标签系统**:
- 支持多标签关联: 如"车险、促销、新客户"
- 按标签筛选查找文案
- 热门标签推荐

**文案使用统计**:
- 记录文案被复制使用的次数
- 按使用频率排序,优先展示高频文案
- 为运营提供文案效果数据

**文案个性化替换**:
- 文案中包含变量占位符: `{name}`, `{phone}`, `{company}`
- 用户使用时自动替换为个人信息
- 示例: "我是{name},专业保险顾问,联系电话{phone}"

## 三、客户邀请模块详细设计

### 3.1 邀请机制概述

#### 3.1.1 功能目标
通过邀请奖励机制,激励代理人拓展下级代理和客户,实现业务快速裂变增长。

#### 3.1.2 邀请流程

```
邀请人 → 生成邀请码/邀请链接 
     ↓
被邀请人 → 通过邀请链接注册 
     ↓
系统绑定 → 建立上下级关系 
     ↓
触发奖励 → 邀请人获得奖励积分/佣金
```

### 3.2 邀请码生成机制

#### 3.2.1 邀请码规则

**生成算法**:
1. 基础规则: 用户ID + 时间戳 + 随机数
2. 编码方式: Base62编码(0-9,a-z,A-Z)
3. 长度控制: 8-10位字符
4. 唯一性保证: 数据库唯一索引约束

**邀请码示例**:
- `A3Bx9Km2`
- `Zy7Pq4W1`

**邀请链接构成**:
```
https://h5.yourdomain.com/invite?code=A3Bx9Km2
```

#### 3.2.2 邀请码管理

**邀请码状态**:
- 未使用: 生成后等待被使用
- 已使用: 被邀请人已注册绑定
- 已过期: 超过有效期(可配置,如30天)

**邀请码数据**:
- 每个用户只有一个永久邀请码
- 首次生成邀请码时创建
- 邀请码可重复使用(多人可使用同一邀请码)

### 3.3 邀请关系绑定

#### 3.3.1 绑定时机
- 新用户注册时填写邀请码
- 通过邀请链接访问H5页面,链接中携带邀请码参数
- 注册完成后立即建立邀请关系

#### 3.3.2 绑定规则
- 每个用户只能被邀请一次,邀请关系不可更改
- 邀请人和被邀请人不能为同一人
- 邀请关系支持多级追溯(A邀请B,B邀请C,C邀请D...)

```java
private String generateInviteCode(Long userId) {
    // 1. 检查用户是否已有邀请码
    InviteCodeDO existCode = inviteCodeMapper.selectByUserId(userId);
    if (existCode != null) {
        return existCode.getCode();
    }
    
    // 2. 生成新邀请码
    String code;
    int retryCount = 0;
    do {
        // 使用用户ID + 时间戳 + 随机数生成
        long timestamp = System.currentTimeMillis();
        int random = ThreadLocalRandom.current().nextInt(10000);
        String source = userId + "" + timestamp + random;
        code = Base62.encode(source.hashCode());
        
        // 检查唯一性
        InviteCodeDO exist = inviteCodeMapper.selectByCode(code);
        if (exist == null) {
            break;
        }
        retryCount++;
    } while (retryCount < 5);
    
    // 3. 保存邀请码
    InviteCodeDO inviteCode = new InviteCodeDO();
    inviteCode.setUserId(userId);
    inviteCode.setCode(code);
    inviteCodeMapper.insert(inviteCode);
    
    return code;
}
```

```java
@Scheduled(cron = "0 0 * * * ?") // 每小时执行
public void updateActivityStatus() {
    LocalDateTime now = LocalDateTime.now();
    
    // 1. 未开始 -> 进行中
    List<ActivityDO> notStarted = activityMapper.selectByStatus(ActivityStatusEnum.NOT_STARTED.getStatus());
    for (ActivityDO activity : notStarted) {
        if (now.isAfter(activity.getStartTime())) {
            activity.setStatus(ActivityStatusEnum.IN_PROGRESS.getStatus());
            activityMapper.updateById(activity);
        }
    }
    
    // 2. 进行中 -> 已结束
    List<ActivityDO> inProgress = activityMapper.selectByStatus(ActivityStatusEnum.IN_PROGRESS.getStatus());
    for (ActivityDO activity : inProgress) {
        if (now.isAfter(activity.getEndTime())) {
            activity.setStatus(ActivityStatusEnum.ENDED.getStatus());
            activityMapper.updateById(activity);
        }
    }
}
```

```java
public AppTeamStructureRespVO getTeamStructure(Long userId, Integer maxLevel) {
    // 1. 查询当前用户信息
    TeamStructureDO root = teamStructureMapper.selectByUserId(userId);
    if (root == null) {
        throw exception(TEAM_MEMBER_NOT_EXISTS);
    }
    
    // 2. 递归查询下级
    AppTeamStructureRespVO result = buildTeamTree(root, 1, maxLevel);
    
    return result;
}

private AppTeamStructureRespVO buildTeamTree(TeamStructureDO node, int currentLevel, int maxLevel) {
    AppTeamStructureRespVO vo = TeamConvert.INSTANCE.convert(node);
    
    // 达到最大层级,不再递归
    if (currentLevel >= maxLevel) {
        return vo;
    }
    
    // 查询直属下级
    List<TeamStructureDO> children = teamStructureMapper.selectByParentId(node.getUserId());
    if (CollUtil.isEmpty(children)) {
        return vo;
    }
    
    // 递归构建子节点
    List<AppTeamStructureRespVO> childrenVos = new ArrayList<>();
    for (TeamStructureDO child : children) {
        AppTeamStructureRespVO childVo = buildTeamTree(child, currentLevel + 1, maxLevel);
        childrenVos.add(childVo);
    }
    
    vo.setChildren(childrenVos);
    return vo;
}
```

**性能优化**:
- 当团队层级较深时,避免一次性查询所有层级
- 建议前端分页加载,每次只展开一级
- 使用Redis缓存团队架构树,减少数据库查询

```java
public void calculateTeamPerformance() {
    LocalDate yesterday = LocalDate.now().minusDays(1);
    
    // 1. 查询所有团队成员
    List<TeamStructureDO> members = teamStructureMapper.selectAll();

    }

private void calculateSelfPerformance(Long userId, LocalDate statDate) {
    // 查询用户在指定日期的订单数据

        .map(OrderDO::getCommission)
```

```java
public List<AppTeamRankingRespVO> getTeamRanking(Integer rankType, Integer topN) {
    LocalDate today = LocalDate.now();
    LocalDate statDate;
    Integer statType;
    
    // 根据排名类型确定统计日期和类型
    switch (rankType) {
        case 1: // 日榜
            statDate = today.minusDays(1);
            statType = StatTypeEnum.DAILY.getType();
            break;
        case 2: // 月榜
            statDate = today.withDayOfMonth(1);
            statType = StatTypeEnum.MONTHLY.getType();
            break;
        case 3: // 季榜
            statDate = today.with(TemporalAdjusters.firstDayOfMonth()).minusMonths(today.getMonthValue() % 3);
            statType = StatTypeEnum.QUARTERLY.getType();
            break;
        case 4: // 年榜
            statDate = today.withDayOfYear(1);
            statType = StatTypeEnum.YEARLY.getType();
            break;
        default:
            throw exception(INVALID_RANK_TYPE);
    }
    
    // 查询排名数据
    List<TeamPerformanceDO> performances = teamPerformanceMapper.selectTopNByDateAndType(
        statDate, statType, topN
    );
    
    // 转换为VO
    List<AppTeamRankingRespVO> result = new ArrayList<>();
    int rank = 1;
    for (TeamPerformanceDO performance : performances) {
        AppTeamRankingRespVO vo = new AppTeamRankingRespVO();
        vo.setRank(rank++);
        vo.setUserId(performance.getUserId());
        vo.setOrderCount(performance.getSelfOrderCount());
        vo.setOrderAmount(performance.getSelfOrderAmount());
        vo.setCommission(performance.getSelfCommission());
        
        // 查询用户基本信息
        MemberUserDO user = memberUserMapper.selectById(performance.getUserId());
        vo.setNickname(user.getNickname());
        vo.setAvatar(user.getAvatar());
        
        result.add(vo);
    }
    
    return result;
}
```

```java
public void updateStudyProgress(Long userId, Long courseId, AppUpdateProgressReqVO reqVO) {
    // 1. 查询或创建学习记录
    UserStudyRecordDO record = userStudyRecordMapper.selectByUserIdAndCourseId(userId, courseId);
    if (record == null) {
        record = new UserStudyRecordDO();
        record.setUserId(userId);
        record.setCourseId(courseId);
        userStudyRecordMapper.insert(record);
    }
    
    // 2. 更新学习数据
    record.setChapterId(reqVO.getChapterId());
    record.setLastPosition(reqVO.getPosition());
    record.setLastStudyTime(LocalDateTime.now());
    
    // 累加学习时长(本次学习时长 = 当前位置 - 上次位置)
    int duration = reqVO.getPosition() - (record.getLastPosition() != null ? record.getLastPosition() : 0);
    if (duration > 0 && duration < 3600) { // 防止异常数据(超过1小时视为无效)
        record.setStudyDuration(record.getStudyDuration() + duration);
    }
    
    // 3. 计算学习进度
    CourseDO course = courseMapper.selectById(courseId);
    int progress = (int) ((record.getStudyDuration() * 100.0) / course.getDuration());
    record.setStudyProgress(Math.min(progress, 100));
    
    // 4. 判断是否完成
    if (record.getStudyProgress() >= 95) { // 观看95%以上视为完成
        record.setIsCompleted(true);
        record.setCompleteTime(LocalDateTime.now());
        
        // 更新课程学习人数
        course.setStudyCount(course.getStudyCount() + 1);
        courseMapper.updateById(course);
    }
    
    userStudyRecordMapper.updateById(record);
}
```

**防刷机制**:
- 限制进度更新频率(如5秒更新一次)
- 检测异常快进行为(短时间内进度跳跃过大)
- 记录播放速度,过快视为无效学习

```java
public AppStartExamRespVO startExam(Long userId, Long examId) {
    // 1. 验证考试是否存在且启用
    ExamDO exam = examMapper.selectById(examId);
    if (exam == null || exam.getStatus() == 0) {
        throw exception(EXAM_NOT_EXISTS);
    }
    
    // 2. 检查重考限制
    List<UserExamRecordDO> history = userExamRecordMapper.selectByUserIdAndExamId(userId, examId);
    if (!exam.getAllowRetake() && !history.isEmpty()) {
        throw exception(EXAM_RETAKE_NOT_ALLOWED);
    }
    
    int retakeTimes = history.size();
    if (exam.getMaxRetakeTimes() != null && retakeTimes >= exam.getMaxRetakeTimes()) {
        throw exception(EXAM_RETAKE_LIMIT_EXCEEDED);
    }
    
    // 3. 查询题目
    List<ExamQuestionDO> questions = examQuestionMapper.selectListByExamId(examId);
    if (CollUtil.isEmpty(questions)) {
        throw exception(EXAM_NO_QUESTIONS);
    }
    
    // 4. 随机打乱题目顺序(如果配置)
    if (exam.getRandomOrder()) {
        Collections.shuffle(questions);
    }
    
    // 5. 创建考试记录
    UserExamRecordDO record = new UserExamRecordDO();
    record.setUserId(userId);
    record.setExamId(examId);
    record.setStartTime(LocalDateTime.now());
    record.setRetakeTimes(retakeTimes);
    userExamRecordMapper.insert(record);
    
    // 6. 构建返回结果(题目不返回正确答案)
    AppStartExamRespVO result = new AppStartExamRespVO();
    result.setRecordId(record.getId());
    result.setExamName(exam.getExamName());
    result.setDuration(exam.getDuration());
    result.setTotalScore(exam.getTotalScore());
    result.setPassScore(exam.getPassScore());
    
    List<AppExamQuestionVO> questionVos = ExamConvert.INSTANCE.convertList(questions);
    // 移除正确答案和解析
    questionVos.forEach(q -> {
        q.setCorrectAnswer(null);
        q.setAnalysis(null);
    });
    result.setQuestions(questionVos);
```

```java
public AppExamResultRespVO submitExam(Long userId, Long recordId, AppSubmitExamReqVO reqVO) {
    // 1. 验证考试记录
    UserExamRecordDO record = userExamRecordMapper.selectById(recordId);
    if (record == null || !record.getUserId().equals(userId)) {
        throw exception(EXAM_RECORD_NOT_EXISTS);
    }
    if (record.getSubmitTime() != null) {
        throw exception(EXAM_ALREADY_SUBMITTED);
    }
    
    // 2. 获取考试信息
    ExamDO exam = examMapper.selectById(record.getExamId());
    
    // 3. 批改试卷
    List<ExamQuestionDO> questions = examQuestionMapper.selectListByExamId(exam.getId());
    Map<Long, ExamQuestionDO> questionMap = questions.stream()
        .collect(Collectors.toMap(ExamQuestionDO::getId, q -> q));
    
    int totalScore = 0;
    List<AppAnswerResultVO> answerResults = new ArrayList<>();
    
    for (AppExamAnswerVO answer : reqVO.getAnswers()) {
        ExamQuestionDO question = questionMap.get(answer.getQuestionId());
        if (question == null) {
            continue;
        }
        
        // 判断答案是否正确
        boolean isCorrect = checkAnswer(question, answer.getUserAnswer());
        int score = isCorrect ? question.getScore() : 0;
        totalScore += score;
        
        // 记录批改结果
        AppAnswerResultVO result = new AppAnswerResultVO();
        result.setQuestionId(question.getId());
        result.setUserAnswer(answer.getUserAnswer());
        result.setCorrectAnswer(question.getCorrectAnswer());
        result.setIsCorrect(isCorrect);
        result.setScore(score);
        result.setAnalysis(question.getAnalysis());
        answerResults.add(result);
    }
    
    // 4. 判断是否通过
    boolean isPassed = totalScore >= exam.getPassScore();
    
    // 5. 更新考试记录
    LocalDateTime submitTime = LocalDateTime.now();
    long duration = Duration.between(record.getStartTime(), submitTime).getSeconds();
    
    record.setSubmitTime(submitTime);
    record.setDuration((int) duration);
    record.setScore(totalScore);
    record.setPassStatus(isPassed ? 1 : 0);
    record.setAnswers(JSON.toJSONString(reqVO.getAnswers()));
    userExamRecordMapper.updateById(record);
    
    // 6. 构建返回结果
    AppExamResultRespVO result = new AppExamResultRespVO();
    result.setRecordId(record.getId());
    result.setScore(totalScore);
    result.setTotalScore(exam.getTotalScore());
    result.setPassScore(exam.getPassScore());
    result.setIsPassed(isPassed);
    result.setDuration((int) duration);
    result.setAnswerResults(answerResults);
    
    return result;
}

// 检查答案是否正确
private boolean checkAnswer(ExamQuestionDO question, String userAnswer) {
    if (userAnswer == null || question.getCorrectAnswer() == null) {
        return false;
    }
    
    switch (question.getQuestionType()) {
        case 1: // 单选
        case 3: // 判断
            return userAnswer.equalsIgnoreCase(question.getCorrectAnswer());
        
        case 2: // 多选
            String[] userAnswers = userAnswer.split(",");
            String[] correctAnswers = question.getCorrectAnswer().split(",");
            Arrays.sort(userAnswers);
            Arrays.sort(correctAnswers);
            return Arrays.equals(userAnswers, correctAnswers);
        
        case 4: // 填空
        case 5: // 简答
            // 简单字符串比较,实际可能需要更复杂的逻辑
            return userAnswer.trim().equalsIgnoreCase(question.getCorrectAnswer().trim());
        
        default:
            return false;
    }
}
```

```java
package cn.iocoder.yudao.module.marketing.util;

/**
 * 邀请码生成工具类
 */
public class InviteCodeUtil {
    
    private static final String BASE62_CHARS = "0123456789abcdefghijklmnopqrstuvwxyzABCDEFGHIJKLMNOPQRSTUVWXYZ";
    
    /**
     * 生成邀请码
     * 
     * @param userId 用户ID
     * @return 邀请码
     */
    public static String generate(Long userId) {
        long timestamp = System.currentTimeMillis();
        int random = ThreadLocalRandom.current().nextInt(10000);
        long source = userId * 1000000 + (timestamp % 1000000) + random;
        return base62Encode(source);
    }
    
    /**
     * Base62编码
     */
    private static String base62Encode(long num) {
        StringBuilder sb = new StringBuilder();
        while (num > 0) {
            sb.append(BASE62_CHARS.charAt((int) (num % 62)));
            num /= 62;
        }
        return sb.reverse().toString();
    }
}
```

### 11.2 二维码生成工具

```java
package cn.iocoder.yudao.module.marketing.util;

import com.google.zxing.BarcodeFormat;
import com.google.zxing.EncodeHintType;
import com.google.zxing.client.j2se.MatrixToImageWriter;
import com.google.zxing.common.BitMatrix;
import com.google.zxing.qrcode.QRCodeWriter;

/**
 * 二维码生成工具类
 */
public class QRCodeUtil {
    
    /**
     * 生成二维码
     * 
     * @param content 二维码内容
     * @param width 宽度
     * @param height 高度
     * @param filePath 保存路径
     */
    public static void generate(String content, int width, int height, String filePath) throws Exception {
        Map<EncodeHintType, Object> hints = new HashMap<>();
        hints.put(EncodeHintType.CHARACTER_SET, "UTF-8");
        hints.put(EncodeHintType.MARGIN, 1);
        
        QRCodeWriter qrCodeWriter = new QRCodeWriter();
        BitMatrix bitMatrix = qrCodeWriter.encode(content, BarcodeFormat.QR_CODE, width, height, hints);
        
        Path path = FileSystems.getDefault().getPath(filePath);
        MatrixToImageWriter.writeToPath(bitMatrix, "PNG", path);
    }
    
    /**
     * 生成带Logo的二维码
     */
    public static void generateWithLogo(String content, int width, int height, 
                                        String logoPath, String outputPath) throws Exception {
        // 1. 生成普通二维码
        generate(content, width, height, outputPath);
        
        // 2. 读取二维码图片
        BufferedImage qrImage = ImageIO.read(new File(outputPath));
        
        // 3. 读取Logo图片
        BufferedImage logoImage = ImageIO.read(new File(logoPath));
        
        // 4. 将Logo绘制到二维码中心
        Graphics2D g = qrImage.createGraphics();
        int logoWidth = width / 5;
        int logoHeight = height / 5;
        int x = (width - logoWidth) / 2;
        int y = (height - logoHeight) / 2;
        
        g.drawImage(logoImage, x, y, logoWidth, logoHeight, null);
        g.dispose();
        
        // 5. 保存
        ImageIO.write(qrImage, "PNG", new File(outputPath));
    }
}
```

### 11.3 海报合成工具

```java
package cn.iocoder.yudao.module.marketing.util;

import javax.imageio.ImageIO;
import java.awt.*;
import java.awt.image.BufferedImage;
import java.io.File;

/**
 * 海报合成工具类
 */
public class PosterUtil {
    
    /**
     * 合成海报
     * 
     * @param templatePath 模板路径
     * @param elements 元素配置
     * @param outputPath 输出路径
     */
    public static void compose(String templatePath, PosterElements elements, String outputPath) throws Exception {
        // 1. 读取模板图片
        BufferedImage template = ImageIO.read(new File(templatePath));
        Graphics2D g = template.createGraphics();
        
        // 2. 设置抗锯齿
        g.setRenderingHint(RenderingHints.KEY_ANTIALIASING, RenderingHints.VALUE_ANTIALIAS_ON);
        g.setRenderingHint(RenderingHints.KEY_TEXT_ANTIALIASING, RenderingHints.VALUE_TEXT_ANTIALIAS_ON);
        
        // 3. 绘制图片元素
        if (elements.getImages() != null) {
            for (PosterImageElement image : elements.getImages()) {
                if (image.getUrl() != null) {
                    BufferedImage img = ImageIO.read(new File(image.getUrl()));
                    g.drawImage(img, image.getX(), image.getY(), image.getWidth(), image.getHeight(), null);
                }
            }
        }
        
        // 4. 绘制文本元素
        if (elements.getTexts() != null) {
            for (PosterTextElement text : elements.getTexts()) {
                drawText(g, text);
            }
        }
        
        // 5. 绘制二维码
        if (elements.getQrcode() != null) {
            PosterQRCodeElement qrcode = elements.getQrcode();
            BufferedImage qrImage = ImageIO.read(new File(qrcode.getUrl()));
            g.drawImage(qrImage, qrcode.getX(), qrcode.getY(), qrcode.getWidth(), qrcode.getHeight(), null);
        }
        
        g.dispose();
        
        // 6. 保存合成后的图片
        ImageIO.write(template, "PNG", new File(outputPath));
    }
    
    /**
     * 绘制文本
     */
    private static void drawText(Graphics2D g, PosterTextElement text) {
        // 设置字体
        Font font = new Font(text.getFontFamily(), Font.PLAIN, text.getFontSize());
        g.setFont(font);
        
        // 设置颜色
        g.setColor(Color.decode(text.getColor()));
        
        // 绘制文本
        String content = text.getValue();
        if (content.length() > text.getMaxLength()) {
            content = content.substring(0, text.getMaxLength()) + "...";
        }
        
        g.drawString(content, text.getX(), text.getY());
    }
}
```
